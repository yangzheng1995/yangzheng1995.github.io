linux 远程拨号

发表于 2016 Sep 17   |   分类于 Linux   |  

服务器：Centos 7.1

远程连接Linux

1.在windows下 可以用putty进行连接 在固定位置输入IP地址，端口号，直接进行连接
2.在linux下使用ssh进行远程连接，例如你的服务器ip是 11.11.11.11

然后输入服务器密码

Linux 远程拨号

先在终端输入

然后输入：pppoe-setup //利用pppoe进行远程拨号
依次输入：

PPPOE user name : ADSL账号

interface : 输入 eth1 (根据你服务器配置的网卡判断，eth0，eth1，eth2)

Do you want link to demand : no (default)

DNS：server (回车)

password : ADSL密码

firewalling: 0
确认信息

设置完成后，可以直接输入 pppoe-start 进行连接，
然后可测试一下是否可以上网

如果ping通的话就说明拨号成功

断开命令： pppoe-stop 查看状态：pppoe-status
